Tuto Js Avanzado Intro JSON - Ejemplo 2 (Javascript Object Notation)

JSON es la abreviatura de Javascript Object Notation y se trata de una forma de almacenar información de forma organizada y de fácil acceso de la forma: Pares clave/valor.

Es un estándar que se utiliza para muchas cosas aparte de Javascript, para Bases de Datos como las MongoDB, incluso en lenguajes de programacion de desarrollo web de backend, como Php, Java, Ruby.
